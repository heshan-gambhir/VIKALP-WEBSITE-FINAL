>>How Our Game Is Different From Existing Games?

Our game is different and unique from other existing games in numerous ways. One such attribute we can categorise is the format of the game. It is certainly not a new format but the storyline and idea behind this project certainly is our own. Our main agenda behind this is to provide the player the knowledge in an interactive way so that one doesn’t feel or think he/she is learning. This has been the agenda for a lot of games that are currently existing but we believe that if a player would be able to relate it to his/her real life it would become more engrossing and the player may use the teachings in reality. Nowadays we are so close to technology that we tend to grasp things faster through such mediums. So, we believe that students will be able to learn and acquire knowledge which the teachings convey in a quicker way. It is rightly said that we learn from experiences and mistakes and keeping this in mind, if any individual plays our game then he/she will be able to make choices in the real world easily because we have already stimulated such a choice in our game and provided insight so as to support the fact why we made that decision. When playing a game we become so engrossed that we tend to forget about how much time has elapsed, which gives us more time to think about situations or events that could be taking place in the future.This is what makes our game stand out from others.
